# 백준 11657 타임머신

이 문제는 그래프 간선이 음수값도 있다. 따라서 다익스트라 알고리즘을 사용할 수 없다.<br>
이 문제를 풀기 위한 핵심 알고리즘은 벨먼포드 알고리즘이다.<br>
총 v개의 노드가 있고 e개의 간선이 있다면, |v-1|번 e개의 간선에 대해 dist[]를 갱신해준다.<br>
마지막으로 한번 e개의 간선에 대해 dist[]를 갱신하려고 할 때 갱신이 되면 음의 사이클이 있다는 것이다.<br>

v-1번 반복해야 하는 이유는 정점 v개를 모두 연결하는 최악의 경우 edge가 최대 v-1개 발생하기 때문이다.<br>
다시 말해 최악의 경우 v-1번 순회를 해야한다.

음의 사이클이 존재하는 경우 -1을 출력하였고 도착지점의 dist가 출발지점의 dist와 간선의 시간을 더한 것보다 작으면<br>
갱신해준다.
